# Support

Use the following channels depending on request type:

- **Bug reports and feature requests**: GitHub Issues
- **Security reports**: private report channel (see `SECURITY.md`)
- **General usage questions**: GitHub Discussions (recommended) or Issues

## When Asking for Help

Please include:

- `@flyrank/dna` version
- Node version
- runtime target (local, CI, Vercel, worker, etc.)
- exact API call and options used
- full error message/stack trace
- minimal reproduction if possible
